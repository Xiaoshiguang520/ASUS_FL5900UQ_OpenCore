<hr>
注意！<br>
1.Opencore5.9支持到MAC10.15.6系统，不支持到最新MAC 11.0，后续会更新。<br>
2.每个人的电脑硬件有些不用，安装系统时候会出现不一样的问题。<br>
3.为了保证OC稳定性，触控板和WIFI默认关闭，安装完成后根据自己配置开启即可。<br>
4.第一次OpenCore引导win10会出现修复微软账户权限提醒，后面就不会出现了。
